About this Project

The current focus, here, is to make some progress towards utility ECMAscript programs. They target a tablet thing and they intend to add functionality on the move.

A part of this views facilities that are available, online. This project uses the Google Cloud to make provide the path to these tools.

Some Ways along this Path are noticed

Deployment using git push takes its time. To releave the pressure from the git 'deployer' the command line is given within a nohup command. The terminal session can be terminated, while the 'deployer' can have its time to do its work. 



 nohup git push &

 tail -f nohup.out
